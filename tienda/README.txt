SB_Tienda is a collection of modules implementing an ecommerce environment, including rudimentary CMS.  It was originally written in Nicaragua for an online hammock store, hence the Spanish filenames.

Modules:

SB_Bajo -- base class
SB_Tienda -- The store environment, displays catalog listings and handles checkout
SB_TiendaDB -- database transactions for the store
SB_TiendaReport -- store reports
SB_Cesto -- the shopping cart
SB_CestoDB -- database transactions for the shopping cart
SB_Caja -- the checkout environment
SB_Arregla -- store management:  product management, order tracking, reports, site content
